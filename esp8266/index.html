Welcome to my Web API for the ESP8266

Please use the following commands to control the ESP

  1. GET http://localhost:3000/changeEspStatus?espStatus=n : Changes the espstatus to n
  2. GET http://localhost:3000/espStatus : Changes the espstatus
  3. GET http://localhost:3000/buttonIsPressed : To inform the server that the button is pressed
